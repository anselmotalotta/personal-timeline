# Configuration Files

This folder contains configuration files and setup scripts:

## Setup Scripts
- `setup_dev_uv.sh` - Development environment setup with UV
- `start_app.sh` - Application startup script
- `test_*.sh` - Testing scripts

## Docker Scripts
- `RESTART_DOCKER.sh` - Docker restart utility
- `teardown.sh` - Environment teardown
- `verify_*_docker_setup.sh` - Docker setup verification

## Configuration Files
- `ingest.conf` - Data ingestion configuration
- `archive/` - Archived configuration files
